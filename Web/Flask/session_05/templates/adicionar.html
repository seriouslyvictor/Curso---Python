GET vs POST — a diferenca essencial

Ambos enviam dados do navegador para o Flask. A diferenca e onde esses dados ficam na requisicao.

GET Sessao 4

Como envia
Dados aparecem na URL depois do ?
URL resultante
/?valor=25&tipo=celsius
Flask le com
request.args.get("chave")
Quando usar
Buscas, filtros, consultas que podem ser marcadas como favoritas

POST Sessao 5

Como envia
Dados ficam no corpo (body) da requisicao, ocultos
URL resultante
/adicionar (sem dados visiveis)
Flask le com
request.form.get("chave")
Quando usar
Criar, modificar, deletar dados. Senhas, votos, envios unicos

Experimento: abra as Ferramentas do Desenvolvedor (F12), va em Network e envie o formulario. Clique na requisicao POST /adicionar e procure a aba Payload ou Form Data. Os dados estao la, no corpo — a URL continuou limpa.
